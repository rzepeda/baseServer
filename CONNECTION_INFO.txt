====================================
TROUBLESHOOTING
====================================

If Claude.ai shows "server not responding" or "connection error":

1. The Cloudflare tunnel URL may have changed. Check:
   - Run 'cloudflared tunnel info' to get current URL
   - Update .env with new CLOUDFLARE_TUNNEL_URL

2. Server may need restart:
   - pkill -f "python -m src"
   - source .venv/bin/activate && python -m src > server.log 2>&1 &

3. Check server is listening:
   - ps aux | grep "python -m src"
   - curl http://localhost:8080/health
